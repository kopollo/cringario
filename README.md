# cringario
## Features
- Two game modes
  - Singleplayer mode
  - Competition mode with other player on one keyboard
- –ê lot of different levels
- Bonus system
- Amazing textures and sprites
- Time, score, health indicators

## Dependencies
- pygame
- pygame_gui
- PyYAML

## Codestyle linters
The project has been checked with the following tools:
- flake8
- mypy
- pydocstyle
